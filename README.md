# IBM Granite for Researchers

## Nagoya University 5-Day Faculty Seminar on Responsible GenAI in Data Science Education

**Author:** Dr. Jeanne McClure, PhD | NC State University Data Science Academy / Ars Innovate  
**Funding:** National Science Foundation, DUE-2313644

---

## Overview

This repository contains hands-on Jupyter Notebook tutorials developed for the Nagoya University 5-Day Faculty Seminar on Teaching with AI. The tutorials guide researchers and educators through using IBM's open-source Granite models for various AI-powered tasks, from conversational AI to agentic workflows.

All tutorials are designed to run in Google Colab with free-tier resources.

---

## Tutorials

### Tutorial 1: IBM Granite Introduction to Conversational AI
**File:** `Tutorial_1_IBM_Granite_Introduction_v2_.ipynb`  
An introduction to IBM Granite models and conversational AI. Learn how to set up and interact with Granite models, understand model parameters, and build basic conversational interfaces.

### Tutorial 2: RAG Under the Hood
**File:** `Tutorial_2_RAG_Under_the_Hood.ipynb`  
A deep dive into how Retrieval-Augmented Generation (RAG) works. Explore the core components of RAG pipelines including document chunking, embedding, vector storage, and retrieval-based question answering.

### Tutorial 3: IBM Granite Docling RAG
**File:** `Tutorial_3_IBM_Granite_Docling_RAG_(shared).ipynb`  
Build a RAG system using IBM Docling for advanced document processing. Learn to parse PDFs and other document formats, create embeddings, and perform intelligent document-based Q&A with Granite models.

### Tutorial 4: Agent-Assisted Workflows with IBM BEE
**File:** `Copy_of_Tutorial_4_Agent_Workflows_IBMBEE(1).ipynb`  
Explore agentic AI workflows using the IBM BEE framework. Learn to build track-specific agentic tools that can autonomously plan, reason, and execute multi-step tasks.

---

## Getting Started

1. Click on any tutorial notebook above
2. Click the "Open in Colab" badge or open the notebook directly in [Google Colab](https://colab.research.google.com/)
3. Follow the instructions in each notebook to set up your environment
4. Each tutorial includes step-by-step guidance and code cells to run

### Prerequisites

- A Google account (for Google Colab access)
- An IBM API key (instructions provided in each notebook)
- No local installation required — all tutorials run in the cloud
